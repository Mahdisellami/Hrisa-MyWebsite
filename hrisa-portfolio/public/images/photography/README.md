# Photography Images

Add your photography here with these filenames:

- **photo-01.jpg** through **photo-12.jpg** (or more)

## Specifications:
- Square aspect ratio recommended (1:1)
- At least 800x800px
- JPG format
- Target file size: 100-300KB each

## Variety suggestions:
- Cityscapes from your travels (Munich, Berlin, Paris, etc.)
- Portraits
- Architecture
- Street photography
- Nature/landscapes
- Cultural moments

## Optimization:
```bash
# Resize and optimize
convert input.jpg -resize 1000x1000^ -gravity center -extent 1000x1000 -quality 85 photo-01.jpg
```

The gallery will automatically display your photos once added!
