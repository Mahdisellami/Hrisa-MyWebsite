# Artwork Images

Add photos of your artwork here with these filenames:

- **art-01.jpg** through **art-09.jpg** (or more)

## Specifications:
- Portrait/vertical aspect ratio recommended (3:4)
- At least 900x1200px
- JPG format
- Target file size: 100-400KB each

## Tips:
- Photograph artwork in good lighting (natural daylight is best)
- Use a flat surface for drawings/paintings
- Minimize shadows
- Ensure colors are accurate
- Take photos straight-on (not at an angle)

## Optimization:
```bash
# Resize maintaining aspect ratio
convert input.jpg -resize 900x1200 -quality 85 art-01.jpg
```

The gallery grid will automatically display your artwork!
