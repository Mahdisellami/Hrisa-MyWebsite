- **fincomp.jpg** - FinComp knowledge graph screenshot (1200x800px recommended)
- **agent-chat-ui.jpg** - agent-chat-ui interface screenshot (1200x800px recommended)

## Tips:
- Use JPG format for photos/screenshots
- Recommended size: 1200x800px (landscape, 3:2 aspect ratio)
- Optimize images to 80% quality for web
- Keep file sizes under 300KB if possible
- Show the most interesting/impressive part of each project

## Quick optimization:
```bash
# Using ImageMagick
convert input.png -resize 1200x800 -quality 80 output.jpg

# Or use online tools:
# - TinyPNG.com
# - Squoosh.app
```
