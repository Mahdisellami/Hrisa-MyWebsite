# Documents Folder

## CV/Resume

Add your CV here as:

**cv.pdf**

The download button on the About page links to `/docs/cv.pdf`. Once you add this file, the download will work automatically.

## Tips:
- Keep filename as `cv.pdf` (lowercase)
- Ensure it's up-to-date with your latest projects
- Target file size: under 2MB
- Include contact information
- Highlight projects featured on this website

---

## Future: Publications

If you have research publications, you can create a `publications/` subfolder here:

```
docs/
├── cv.pdf
└── publications/
    ├── paper-title-2024.pdf
    └── another-paper-2023.pdf
```
